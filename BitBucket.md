# Introduction #

[BitBucket](http://bitbucket.org) is a public source code hosting service UsingMercurial for its distributed version control system ([DVCS](http://en.wikipedia.org/wiki/Distributed_revision_control)).  It is being used by Core Plot for the [WWDC 2009 code-a-thon](http://code.google.com/p/core-plot/wiki/WWDCStartPage).

# Things You Should Know #

  * How to sign up: http://bitbucket.org/account/signup/
  * Where Core-Plot lives: http://bitbucket.org/drewmccormack/core-plot-wwdc/
  * How to share a repository: http://bitbucket.org/help/Collaborating
